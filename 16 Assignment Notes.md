## Learned Today

- Difference between a normal program, automation, and an AI Agent.
- An AI Agent works toward a goal and can coordinate multiple components.
- An LLM is usually one component inside an AI Agent, not the entire agent.
- The term "agent" comes from acting on behalf of the user to achieve a goal.

### Key Idea

An AI Agent is not defined by the number of tools it uses. What makes it an agent is that it can decide how and when to use available tools to achieve the user's goal, instead of blindly following a fixed sequence of instructions.
