# AI Agent

## what is an AI Agent ?

An AI Agent is a system that take a goal and decides which action to perfrom to acheive it . unlike a normal program that follows fixed instructions , an AI Agent can choose different actions based on the situation and may use external tools like search engines or APIs

## Difference from a Normal Program

- Normal program: Executes predefined instructions.
- AI Agent: Chooses actions based on the goal to complete  the goal .

## Real-world Example

A navigation app doesn't just follow one fixed route. If a road is closed, it finds another path to reach the destination. Similarly, an AI Agent adapts its actions while working toward a goal.

# Why is it called an Agent?
An AI Agent is called an agent because it acts on behalf of the user to achieve a goal. Instead of following one fixed instruction, it coordinates multiple components like an LLM, search tools, and other processing steps to decide how to complete the user's request.

# LLM Limitations

## LLM is not a database

An LLM stores patterns learned during training. It does not automatically know information created after its training. To answer questions about recent information, it needs external tools such as search engines, databases, or APIs.
