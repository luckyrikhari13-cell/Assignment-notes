# What is LangChain?

## Definition

LangChain is a framework for building applications powered by Large Language Models (LLMs). It provides reusable components that can be connected together to create AI workflows.

## Is LangChain an LLM?

No.

LangChain does not generate answers itself. It coordinates components such as prompts, LLMs, tools, retrievers, parsers, and memory.

## Why do we need LangChain?

Without LangChain, developers would manually connect many functions together. LangChain provides reusable building blocks that make AI applications easier to organize, modify, and maintain.

## Key Idea

LangChain treats many different components as "Runnables" so they can be composed into flexible pipelines.
