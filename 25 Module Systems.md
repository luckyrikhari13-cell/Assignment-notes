# Module Systems

## Module

A module is a file that can import and export code.

---

## CommonJS

Uses

```js
require()
module.exports
```

---

## ES Modules (ESM)

Uses

```ts
import
export
```

---

## package.json

```json
"type": "module"
```

tells Node to treat files as ES Modules.

---

Modern libraries like LangChain use ES Modules.
